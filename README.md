## Commit Convention
- DOCS: 문서 작성 및 수정 작업(README 등)
- FEAT: 새로운 기능 추가 작업
- FIX: 에러 및 버그 수정, 기능 수정 작업
- REFACTOR: 코드 리팩토링 작업
- CHORE: 빌드 업무 및 패키지 매니저 수정 작업 등의 작업
- TEST: 테스트 관련 작업

## Coding Convention
